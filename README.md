# React Template
React starte kit with, Redux, ES7, CSS Modules.

## Get up and running
With `node` installed run
```bash
npm install
```

## Test
To test in development
```bash
npm run test-watch
```

To run one go
```bash
npm test
```

## Development
When developing run
```bash
npm run dev
```
and the app will auto refresh on every save

## Production
```bash
npm start
```
